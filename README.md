# Find-a-protein
An application that uses a deep learning approach to find protein subcellular localisation data from scientific literature.

Tested on 
- Python 3.X
- Pytorch 0.4
- Pandas 0.20.3
- Numpy 1.13.3
- Gensim 3.4.0
- Scikit-learn 0.19.1
- Nltk 3.2.4

python3 main.py [optional parameters]
